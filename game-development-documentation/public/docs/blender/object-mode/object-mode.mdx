## Updating Object Origin

### Object Mode Version
1. Enable **Options > Affects Only: Origins**
![](./assets/only-origins.png)

2. You can now move the origin as you would an object. Using **G** to grab
![](./assets/changing-object-origin.gif)

### Edit Mode Version
1. To edit multiple objects origin points at once we can do this in edit mode. First select the **Show Overlays** button
![](./assets/show-overlays.png)

2. Ensure that **Origins** is enabled so we can see 
![](./assets/enable-origins.png)

3. **Shift-click** the objects you want to edit
![](./assets/shift-click-objects.png)

4. Enter **Edit Mode** and press **A** to select all the vertices.
![](./assets/go-into-edit-mode.png)

5. Use grab mode **G** to move all the vertices. Essentially you are moving the objects up while the origins stay in place
![](./assets/edit-multiple-object-origins.gif)

### Move Origin Using Geometry

1. Right-Click on your object. Then select **Set Origin > Origin to Geometry**
![](./assets/origin-to-geometry.gif)

2. You can also select **Set Origin > Geometry to Origin** which performs the inverse task
![](./assets/geometry-to-origin.gif)

## Select Linked

1. Press **Shift + L** to open the Select Linked menu or go to **Select > Select Linked**
![](./assets/select-linked-menu.png)

2. Each select type serves different purposes
![](./assets/select-linked-buttons.png)

### Select Linked Menu
| Type | Description | Image |
|----------|-------------|-------|
| **Object Data** | Selects all objects that share the same mesh edits | ![](./assets/select-linked-objects.gif) |
| **Material** | Selects objects that have shared materials | ![](./assets/select-linked-materials.gif) |

### Unlink Object (Make Single User)
1. Select Object
![](./assets/select-object.png)

2. Go to **Object > Relations > Make Single User > Object & Data**
![](./assets/make-object-single-user.png)
