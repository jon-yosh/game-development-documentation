## Selecting & Moving üéØ
| Action                | Shortcut     |
|-----------------------|--------------|
| Select tool           | `S` then press |
| Move in small steps   | **Arrow keys**   |
| Move in bigger steps  | **Shift + Arrow** |
| Duplicate object      | **Ctrl + D**     |
| Delete object         | **Delete**       |

### While In Select Mode
| Action                | Image     |
|-----------------------|--------------|
| `S` again to Rotate   | ![](./assets/moving-and-rotating-objects.gif) |
| **Shift + Click** origin to reset | ![](./assets/shift-click-origin.gif)
| While Dragging press **Space** to copy | ![](./assets/space-to-copy.gif)

### While Drawing
| While Drawing...                | What It Does                              | Images |
|---------------------------------|-------------------------------------------|--------|
| Hold `Ctrl`                     | Lock proportions (square/circle) or snap angles | ![](./assets/shift-modifier.gif) |
| Hold `Shift`                    | Draw from the center instead of a corner  | ![](./assets/ctrl-modifier.gif) |
| Hold `Ctrl + Shift` together    | Perfect shape, drawn from center          | ![](./assets/ctrl-and-shift-modifier.gif) |
