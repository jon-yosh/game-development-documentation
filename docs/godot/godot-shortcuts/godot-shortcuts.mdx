# Godot Shortcuts Reference

## Editor Navigation
| Shortcut | Action |
|----------|---------|
| **CMD+P** | Quick Open (find any file/scene quickly) |
| **CMD+Shift+P** | Command Palette |
| **CMD+Shift+O** | Open Scene (find any scene quickly) |
| **CMD+Alt+O** | Open Script (find any script quickly) |

## Scene/Node Management
| Shortcut | Action |
|----------|---------|
| **CMD+A** | Add Node |
| **CMD+D** | Duplicate Node |
| **CMD+Del** | Delete Node |
| **CMD+G** | Group Selected Nodes |
| **CMD+Shift+G** | Ungroup Nodes |
| **CMD+Shift+F**\*\* | Move Camera to View |
| **CMD+Y**\*\* | Change Node Type |

## Viewport Controls
| Shortcut | Action | Images |
|----------|--------|--------|
| **Middle Mouse** or **Option + One-finger trackpad** | Orbit |
| **Mouse Wheel** or **Ctrl + One-finger trackpad** | Zoom in/out |
| **Shift + Middle Mouse** or **Shift + One-finger trackpad** | Pan |
| **F** | Frame selected node | ![](./assets/f-to-frame.gif) |
| **Shift + F** | Freelook mode | ![](./assets/freelook-mode.gif) |
| **Shift+G** | Toggle grid snap | ![](./assets/reposition-using-collisions.gif) |
| **Shift+S** | Toggle smart snap | 

## Script Editor
| Shortcut | Action |
|----------|---------|
| **CMD+B** | Run project |
| **CMD+R** | Run current scene |
| **F7** | Step into (debugging) |
| **F8** | Step over (debugging) |
| **F9** | Toggle breakpoint |
| **Ctrl+/** | Toggle comment |
| **Ctrl+Shift+/** | Block comment |
| **Ctrl+I** | Auto-indent |
| **Ctrl+J** | Join lines |

## File Management
| Shortcut | Action |
|----------|---------|
| **Ctrl+Shift+S** | Save all open scenes |
| **Ctrl+Alt+S** | Save scene as |
| **Ctrl+Tab** | Switch between open scripts |
| **Alt+Left/Right** | Navigate script history |
| **Ctrl+H** | Replace in script |
| **Ctrl+Shift+H** | Replace in files |

## Transform Tools (2D/3D)
| Shortcut | Action |
|----------|---------|
| **Q** | Select tool |
| **W** | Move tool |
| **E** | Rotate tool |
| **R** | Scale tool |
| **T** | Transform tool (2D only) |

## Setup Shortcuts: *Move Camera to View* as example

![Editor Settings](./assets/editor-settings.png)

1. Open up **Editor Settings...** with `Command+,`

![Shortcuts Window](./assets/shortcuts-window.png)

2. View the **Shortcuts** to change keybinds

![Align with View Shortcut](./assets/align-with-view.png)
3. Create `Shift+Command+F` as the shortcut for both: **Align Rotation with View** and **Align Transform with View**
